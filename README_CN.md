# HypeRate Desktop Overlay (EXE版)

这是一个专为 [HypeRate.io](https://hypeRate.io) 设计的极简桌面心跳挂件。它能够自动获取 Token，并完美覆盖在大多数全屏游戏（如《三角洲行动》）之上。
**注意**：本挂件需与已安装 HypeRate 的智能穿戴设备配合使用，设备负责采集并通过 HypeRate 提供实时心率数据。

## ✨ 主要功能
* **全自动化**：无需手动抓包，只需输入 HypeRate 短 ID (如 `1F01`) 即可使用。
* **游戏覆盖**：支持“全屏优化”模式的游戏置顶显示。
* **鼠标穿透**：锁定后，鼠标可直接穿透挂件点击游戏，防止误触。
* **极简托盘**：无任务栏图标，所有操作通过系统托盘（右下角）完成。

## 🚀 使用方法
1. **启动设备并记录 ID**：启动智能穿戴设备上的 HypeRate 应用，并记录下设备的 HypeRate ID（例如 `1F01`）。
2.  **运行**：双击 `HypeRateOverlay.exe`。
3.  **设置 ID**：首次运行时输入您的 HypeRate ID。
4.  **移动**：按住心跳数字即可拖动位置（位置会自动保存）。
5.  **锁定**：在任务栏右下角找到 **红色爱心 (❤)** 图标，右键选择 **"锁定穿透"**。

## ⚙️ 托盘菜单 (右键 ❤ 图标)
* **🔒 锁定穿透 (Click-through)**：
    * [开启]：挂件变为半透明，鼠标可穿透点击后方窗口（游戏时推荐）。
    * [关闭]：挂件恢复不透明，可进行拖拽移动。
* **🚀 开机自启 (Start at Login)**：随 Windows 自动启动。
* **🔄 重置 ID**：更换 HypeRate ID。
* **❌ 退出**：关闭程序。

## ⚠️ 常见问题
* **配置文件**：保存在 `C:\Users\你的用户名\HypeRateOverlay\config.json`。
* **游戏显示**：如果挂件在游戏中消失，请尝试将游戏设置为 **"无边框窗口 (Borderless)"** 模式。(如英雄联盟， CS2)
